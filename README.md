# Fondo Frontend

> Your AI CFO. Built for founders like you.

The frontend for [Fondo](https://github.com/ishaanthakur25/Fondo-Backend) — an AI financial agent built specifically for young nonprofit founders and student org leaders.

## Live App
🔗 [Try Fondo →](https://ai-fondo.lovable.app)

## Tech Stack
- React + TypeScript
- Tailwind CSS
- TanStack Router

## Backend
The AI backend is built with Python + FastAPI and deployed on Railway.
See the backend repo: [Fondo-Backend](https://github.com/ishaanthakur25/Fondo-Backend)
